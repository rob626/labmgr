Uninstall Symantec Endpoint Protection with the CleanWipe utility

^Back to top

You can use several methods to uninstall the Symantec Endpoint Protection product components. When the usual uninstall methods have been unsuccessful, such as using the Windows Control Panel, you can use the CleanWipe tool to remove a component.

Warning:

Symantec Technical Support does not recommend using CleanWipe the first time you have uninstallation trouble. You should only use CleanWipe as a last resort when the usual uninstallation methods are unsuccessful.

You can contact Technical Support directly if you need help with the CleanWipe tool.

To uninstall Symantec Endpoint Protection with the CleanWipe utility

  1. Double-click Cleanwipe.exe, and click Next.
  2. Accept the license agreement and click Next.
  3. Select the Symantec products you want to remove and click Next twice.
  4. When the tool finishes running, you may be prompted to restart the computer.

    After the computer restarts, CleanWipe reopens and continues to run.

  5. Click Next.
  6. Click Finish.

    The Symantec products you selected are now uninstalled.
